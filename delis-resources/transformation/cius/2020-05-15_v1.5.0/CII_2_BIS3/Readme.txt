
PEPPOL BIS-Billing to OIOUBL conversion
---------------------------------------
The general mapping specification file is located on GitHub: https://github.com/OpenPEPPOL/peppol-bis-invoice-3/tree/master/mapping
(version c0689fc on 7 Oct 2019).


1.0 Purpose
---------------------
Transformation from CII to BIS 3.0 syntax for invoice and credit note.


2.0 Content
-----------
This folder contains:
    XSLT files (.xslt) for converting invoice and credit note,
    Excel mapping documentation (.xlsx) for both file types.


3.0 Release Notes
-----------------
Fix specific error in CII_2_BIS3 conversion (related to AccountingCustomerParty/Party/EndpointID).
